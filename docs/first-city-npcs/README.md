# NPC первого города

Отдельные лор-профили NPC первого города после стартовой арки. Эти файлы сделаны как будущая основа для `character.md`-подобных профилей, но **не подключены к игре** и не являются runtime NPC.

- [Арден Копченый](./arden_kopcheny.md) - хозяин таверны.
- [Мира Вельт](./mira_velt.md) - администратор гильдии авантюристов.
- [Бранн Ровель](./brann_rovel.md) - глава гильдии авантюристов.
- [Тесса Рыжая Метка](./tessa_ryzhaya_metka.md) - разведчица.
- [Корвин Латник](./korvin_latnik.md) - молодой щитоносец.
- [Дориан Скоба](./dorian_skoba.md) - торговец снаряжением.
- [Сельма Горьколист](./selma_gorkolist.md) - алхимик и травница.
- [Варрик Тихий Молот](./varrik_tikhy_molot.md) - кузнец-ремонтник.
- [Лисандр Пыльная Линия](./lisandr_pylnaya_liniya.md) - картограф и продавец сведений.
- [Нэра Сухарь](./nera_sukhar.md) - торговка провиантом.
